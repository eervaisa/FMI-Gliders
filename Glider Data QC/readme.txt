qartod_qc.ipynb
    Contains basics for performing quality control tests on, and storing the results in, a netCDF
    file created by the pyglider package.

    Notable features:
        - List of all possible tests, explanations on what they do and what parameters they use
        - Examples of creating test configurations, performing tests and saving results
        - A section performing the tests and storing results based on a VOTO dataset
